# python-university-system


--Overview

This project is a  Python program designed to manage a university system.
The system demostrates basic programming principles like object-oriented programming, file operations, and exception handling.

--Purpose
		Add and manage students and instructors
		Create and manage courses
	  Enroll students in courses
		Store and retrieve data using files

--Features

		Student Management (Add, View, Search)
		Instructor Management
		Course Management
		Enrollment System
		File Handling (Save & Load data)
		Error Handling (Invalid inputs handled properly)

--Example Usage
		Add a new student
		Create a course
		Assign an instructor
		Enroll a student into a course
		Save and load data
